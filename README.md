# French NCLC 7 Tracker

一个静态交互式进度追踪网页，用于执行 6 个月 TEF Canada / NCLC 7 法语学习计划。

## 功能

- 180 天学习热力图。
- 每日任务打卡。
- 听、说、读、写、词汇、语法、发音、复盘分钟记录。
- 每日笔记、最大错误、明日优先级。
- TEF 听力、阅读、写作、口语模考状态记录。
- 阶段门槛检查。
- 本地自动保存，支持 JSON 导出和导入。

## 使用方式

直接打开 `index.html`，或部署到 GitHub Pages。

数据保存在当前浏览器的 `localStorage` 中。换电脑或清浏览器数据前，请先点击“导出”保存 JSON。

## GitHub Pages

这个 repo 是纯静态站点，不需要构建步骤。GitHub Pages source 可以直接设置为：

- Branch: `main`
- Folder: `/`
